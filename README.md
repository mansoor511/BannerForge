# BannerForge
A professional tool to generate hacker-style banner.
# 🚀 نام ابزار شما (مثلا BannerForge)

📌 **توضیح کوتاه:** این ابزار برای ساخت بنرهای حرفه‌ای و هکری ساخته شده است.

## ✨ قابلیت‌ها
- ساخت بنر با نام دلخواه
- بیش از ۳۰ فونت متفاوت
- قابلیت ذخیره بنر در فایل

## 🛠️ نحوه نصب و اجرا
برای استفاده از این ابزار، کافی است دستورات زیر را در ترمینال کالی لینوکس خود وارد کنید:

```bash
git clone https://github.com/[USERNAME_شما]/[REPO_NAME_شما].git
cd [REPO_NAME_شما]
pip3 install -r requirements.txt
python3 bannerforge.py
